## 🚀 How to Use

### For Teachers

1. **Start Server**: Double-click `start-server.bat`
2. **Login**: Select "Teacher" role
3. **Option A - Demo Mode**: Click "Enter Demo Mode" for solo building
4. **Option B - Create Room**: Set up a classroom project
5. **Environment**: Press 1-8 to switch planets
6. **Blueprint**: Press Shift+B for technical view

### For Students

1. **Login**: Select "Student" role
2. **Join Room**: Click a room or enter code
3. **Build**: Use tools and blocks to create
4. **Chat**: Communicate with team
5. **Submit**: Click "Submit Work" when done
