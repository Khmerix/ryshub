# BrickForge Pro

A school-ready 3D building game with 50+ block types, built with Godot 4.

![BrickForge Pro Screenshot](screenshot.png)

## Features

- **50+ Block Types** organized in 5 category tabs
- **5 Tools**: Build, Paint, Erase, Select
- **8 Environments**: Earth, Mars, Moon, Desert, Jungle, Island, Arctic, Volcanic
- **Undo/Redo** support
- **Save/Load** builds as JSON
- **Full source code** for learning GDScript

## Quick Start

1. Open `project.godot` in Godot 4.x
2. Press F5 to run
3. Build something awesome!

## Controls

| Key | Action |
|-----|--------|
| Left Click | Place/Erase/Paint/Select |
| Right Click + Drag | Pan camera |
| Middle Click + Drag | Rotate camera |
| Scroll | Zoom |
| 1-4 | Switch tools |
| R | Rotate block |
| G | Toggle grid |
| Ctrl+Z | Undo |
| Ctrl+Y | Redo |
| F1 | Help |

## Block Categories

- **Blocks**: Cube, Slope, Wedge, Cylinder, Sphere, Arch, Corner, Plate, Stair, Tube
- **Arch**: Wall, Window, Door, Fence, Pillar, Beam, Brick, Panel, Floor, Gate
- **Nature**: Trees, Rocks, Bushes, Flowers, Cactus, Mushroom
- **Mat**: Wood, Stone, Concrete, Sand, Dirt, Leaves
- **FX**: Metal, Glass, Neon, Glow, Water, Lava, Ice, Snow
