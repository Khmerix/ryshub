# BrickForge Pro - Student Quick Start

## Getting Started (2 minutes)

1. **Launch the game** - Double-click the BrickForge icon
2. **Select a block** - Click any button in the Block Panel (right side)
3. **Build!** - Left-click on the ground to place blocks

## Building Your First Creation

### Step 1: Build a Floor
- Click the **Plate** block (in Blocks tab)
- Click repeatedly on the ground to place plates
- Make a 5x5 area

### Step 2: Add Walls
- Click **Wall** (in Arch tab)
- Place walls around your floor
- Press **R** to rotate walls

### Step 3: Add a Roof
- Click **Slope** (in Blocks tab)
- Place slopes on top of walls

### Step 4: Decorate
- Go to **Nature** tab
- Add trees and rocks around your house

## Cool Things to Try

### Build a Castle
- Use **Stone** blocks for walls
- Add **Pillar** for towers
- Use **Arch** for gates

### Build a Garden
- Use **Dirt** for paths
- Add **Flower**, **Bush**, **Tree**
- Use **Water** for a pond

### Build a Spaceship (Mars world)
- Press the **World** dropdown and select Mars
- Use **Metal** and **Steel** blocks
- Add **Neon** lights
- Use **Glass** for windows

### Build Underwater
- Select **Island** world
- Use **Ice** and **Glass** blocks
- Add blue lighting with **Glow** blocks

## Pro Tips

1. **Grid Snap** - Keep this ON for straight lines
2. **Rotate** - Press R to rotate blocks before placing
3. **Undo** - Ctrl+Z if you make a mistake
4. **Copy Colors** - Use Paint tool (2) to copy colors from existing blocks
5. **Size Buttons** - Try S (small), M (medium), L (large), XL (huge)

## Keyboard Shortcuts

```
1 - Build tool (place blocks)
2 - Paint tool (change colors)
3 - Erase tool (delete blocks)
4 - Select tool (select blocks)

R - Rotate block
G - Toggle grid
Ctrl+Z - Undo
Ctrl+Y - Redo
F1 - Help
```

## Saving Your Work

1. Click **Save** button (bottom right)
2. Your build is saved automatically
3. Click **Load** to continue later

## Share Your Build

1. Find your saved file at:
   - Windows: `%APPDATA%\Godot\app_userdata\BrickForge Pro\`
   - Mac: `~/Library/Application Support/Godot/app_userdata/BrickForge Pro/`
   - Linux: `~/.local/share/godot/app_userdata/BrickForge Pro/`

2. Copy the `build.json` file
3. Share with friends!
